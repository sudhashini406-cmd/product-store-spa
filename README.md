Product Store SPA

A simple Single Page Application (SPA) that displays products from the Fake Store API. Built using React.

Features

Fetches product data from https://fakestoreapi.com/products

Displays product list

Shows product details

Fully responsive UI

Tech Stack

React

JavaScript

Axios / Fetch

CSS / Tailwind (if used)

How to Run

Clone the repository:

git clone https://github.com/sudhashini406-cmd/product-store-spa.git


Navigate into the project:

cd product-store-spa


Install dependencies:

npm install


Start the project:

npm run dev
